# Sistema de Cadastro de Pacientes – Clínica Vida+

Este projeto foi desenvolvido como parte do Projeto Integrado – Análise e Desenvolvimento de Sistemas.  
O objetivo é criar um sistema simples para cadastrar pacientes e gerar estatísticas sobre os registros da clínica.

## Funcionalidades
- Cadastro de pacientes (nome, idade, telefone)
- Estatísticas
  - Total de pacientes cadastrados
  - Idade média dos pacientes
  - Paciente mais novo e mais velho
- Busca de paciente por nome
- Listagem de todos os pacientes cadastrados

## Tecnologias Utilizadas
- Python 3
- Estruturas de dados listas e dicionários
- Estruturas de controle (loops, condicionais)
- Funções com modularização de código
